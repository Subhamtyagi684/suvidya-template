Why "Suvidya" ?

The decision process at the organizational level is delayed or slow due to the absence of timely and appropriate information about the various functioning of the school. The single-integrated platform provides extensive data consolidation from school functioning, making relevant data available for effective process implementation.
It is often observed that teachers spend less time teaching and more in managing the daily routines of the school like .

  • Write homework into every student's notebook.
  • For marking students' attendance, teachers write every student's name and keep them Present or Absent take so much time every day.
  • Sharing events or notices to every parent.
  • Sharing student progress with parents.
  • Exam and result management

The "Suvidya" advantage

The platform automates organization back-office processes related to technology, services, and human resource data. It also aims to automate organization paperwork by digitizing school data. It provides seamless information flow between teachers, principals, non-teaching staff and parents, thereby providing an efficient and professional work culture. It ensures transparency in the working styles of the school and offers better collaborations between the school and parents. It will also improve the school branding and, therefore, ensure visibility.

Improvement in teaching

The platform ensures more time in teaching by automating all the routine activities of the teacher, thus ensuring that the teachers improve their skills and pedagogy to impart education scientifically. It provides in-depth analyses of the student's academic and mental growth. The software will assist in developing collaborations beyond classrooms.

Improvement in Administration

  • Managing teaching and non-teaching staff leaves.
  • Automation of all the fees related tasks, library management, inventories etc.
  • The cloud application will give users maximum availability, data security and API integration like payment gateway services to accept fees or grants or support online.

The deliverable

  • A "Vidyabharti" mobile app developed using "Suvidya" education governance platform with all the features
  • A "Vidyabharti" web portal developed using "Suvidya" education governance platform with the administrative-level services and features

Essential features

The overall functioning of the schools programmed on the "Suvidya" education governance platform can be divided into nine significant sub-branches

  1. Admission
  2. Curriculum and Time-table
  3. Attendance
  4. Library
  5. Assignments, tests and evaluation
  6. Report cards
  7. Finance
  8. Transport
  9. Hostel

The Teacher's access will have the following features

  • Access to the defined syllabus (with follow-up feature to trace course coverage)
  • Per class attendance (through mobile or biometric). Attendance monitoring and messaging system
  • One-touch assignment sending to all students
  • One-touch exam module
  • Teachers notes (if the teacher wants, can share with students)

The Student's access will have the following features

  • Access to all class students network (for sharing homework assignments)
  • Assignments uploaded by the teacher
  • Access to the syllabus (along with class coverage, so that it can be accessed if a student is on leave)
  • Online class module
  • Leave management system

The parents' access will have the following features

  • Access to parent network (of class)
  • Access to the syllabus (along with class coverage, so that it can be accessed if a student is on leave)
  • Assignments uploaded by the teacher
  • Time-table reminders for class
  • Transport access (if GPS is available on transport)

The headmistress access will have the following features

  • Class status monitoring
  • Access to teacher network
  • Access to the syllabus (along with class coverage)
  • Exam planner
  • One-touch mail to teacher and administration
  • Fee's status of all the students
  • One-touch message sending to parents, students and administration

The Librarian access will have the following features

  • Book management system
  • Class time-tables
  • Periodicals/e-newspaper module (for sending it to students and teachers)
  • Leave management system

The Vice-principal access will have the following features

  • Class status monitoring
  • Access to teacher network
  • Access to the syllabus (along with class coverage)
  • Exam planner
  • One-touch mail to teacher and administration
  • Fee's status of all the students
  • Leave management system

The Principal access will have the following features

  • Class status monitoring
  • Access to teacher network
  • Access to the syllabus (along with class coverage)
  • Exam planner
  • One-touch mail to teacher and administration
  • Fee's status of all the students
  • Leave management system
  • One-touch message sending to parents, students and administration
  • One-touch student analytics module (to monitor each student progress)
  • One-touch teacher/class analytics module (to monitor each class/teacher)
